# Rule-Based AI Chatbot

## Project Title
Rule-Based AI Chatbot - Artificial Intelligence Project 1

## Description
This project is a simple rule-based AI chatbot created using Python. The chatbot responds to predefined user inputs using control flow logic and dictionary-based response mapping. It also handles unknown inputs using a fallback response and runs continuously until the user enters an exit command.

The chatbot uses basic rule-based AI logic, where responses are selected according to fixed conditions and predefined user inputs.

## Features
- Handles basic greetings
- Responds to predefined user inputs
- Uses input sanitization with `lower()`, `strip()`, punctuation removal, and extra space handling
- Uses dictionary-based response lookup
- Provides fallback response for unknown inputs
- Runs in a continuous loop
- Supports exit commands such as `bye`, `exit`, `quit`, and `stop`
