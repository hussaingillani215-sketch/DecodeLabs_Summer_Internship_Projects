# AI Recommendation Logic - Project 3

## Project Overview

This project was created as part of the DecodeLabs Artificial Intelligence Internship Program.

The goal of this project is to build a simple AI-based recommendation system that takes user interests as input and recommends the most relevant items based on similarity logic.

This project uses a content-based filtering approach, where recommendations are generated by comparing user preferences with item tags and descriptions.

## Project Objective

The main objective is to:

- Take user input in the form of interests or preferences
- Match user preferences with available items
- Calculate similarity between the user profile and item data
- Display the top recommended items

## Recommendation Technique Used

This project uses:

- TF-IDF Vectorization
- Cosine Similarity
- Content-Based Filtering

## How It Works

The recommendation system follows these steps:

1. Load or create a sample recommendation dataset
2. Accept user interests as input
3. Combine item tags, category, and description into a single feature column
4. Convert text data into numerical vectors using TF-IDF
5. Calculate cosine similarity between the user profile and item vectors
6. Sort items based on similarity score
7. Display the top recommended items
